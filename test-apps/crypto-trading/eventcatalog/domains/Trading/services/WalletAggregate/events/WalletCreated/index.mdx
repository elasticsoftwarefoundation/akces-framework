## Overview

<Admonition type="info">
  This event is emitted when a new cryptocurrency wallet is created for a user.
</Admonition>

The `WalletCreatedEvent` signals the successful creation of a new wallet for a specific user and cryptocurrency.

## Event Structure

| Field | Type | Description |
|-------|------|-------------|
| `userId` | String | The user ID that owns the wallet (aggregate identifier) |
| `walletId` | String | The unique identifier for the newly created wallet |
| `currency` | String | The type of cryptocurrency for this wallet (e.g., "BTC", "ETH") |
| `initialBalance` | BigDecimal | The initial balance of the wallet |
| `createdAt` | Timestamp | When the wallet was created |

## Producers

This event is produced by:
- **WalletAggregate**: When handling a CreateWallet command

## Consumers

This event is consumed by:
- **WalletQueryService**: To maintain read models of wallets
- **NotificationService**: To notify users of wallet creation
